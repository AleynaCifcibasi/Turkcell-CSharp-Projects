# Kitaplık Projesi

Bu proje, Turkcell Algoritma ve Programlama eğitimi kapsamında C# kullanılarak geliştirilmiş bir konsol uygulamasıdır. Kullanıcıların kitap ekleme, listeleme, arama ve silme işlemlerini gerçekleştirebildiği basit bir kitaplık yönetim sistemidir.

## Edindiğim Kazanımlar

- C# programlama dilinde temel ve orta seviye uygulamalar geliştirdim.
- Koşul ifadeleri (`if-else`, `switch-case`) ve döngü yapılarını etkin şekilde kullandım.
- Metot (fonksiyon) oluşturma ve kullanma pratiği kazandım.
- Diziler ve koleksiyonlar üzerinde işlem yapmayı öğrendim.
- Dosya okuma ve dosyaya veri yazma işlemlerini uyguladım.
- Menü tabanlı konsol uygulaması geliştirme deneyimi edindim.
- Hata kontrolü ve kullanıcıdan güvenli veri alma konularında pratik yaptım.
- Git ve GitHub kullanarak proje versiyonlama sürecini deneyimledim.

## Kullanılan Teknolojiler

- C#
- .NET
- Console Application
- Git & GitHub
